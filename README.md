**Purpose:**  
Providing beautiful, free poker tracking software for everyone, precisely for casual players.

## Tech Stack

- **Frontend:** React
- **Styling:** Tailwind CSS
- **Backend & Database:** Supabase
- **Server:** Node.js with Express
